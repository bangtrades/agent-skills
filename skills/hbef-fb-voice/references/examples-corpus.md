## Voice diagnostic — "is this HBEF?"

When evaluating whether a draft matches HBEF voice, check:

| Check | Pass condition |
|---|---|
| Pronouns | First-person plural ("we," "our") ≥ 3 times |
| Specificity | At least one exact number (dollar, %, date, time) |
| Named people | At least one named volunteer/donor/partner if archetype calls for it |
| Theme alignment | If themed event: theme term used at least 3× across the post |
| Emoji count | 0–2 emoji; 3 only on event-night excitement |
| Hashtag count | 0–5, all at end on a dedicated line |
| CTA | One clear, UPPERCASE button-style action |
| Length | Visible-before-fold < 480 chars |
| HBEF vs HVPTO | Not confused if both are relevant |
| Tax language | Present if explicit donation ask |
| Forbidden words | No "leverage," "synergy," "blessed," "thoughts and prayers," "click here" |

If a draft fails 2+ checks, rewrite. Start from a formula in `formula-library.md`.
