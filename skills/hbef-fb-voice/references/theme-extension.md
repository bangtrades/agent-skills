# HBEF Theme Extension — The Signature Move

HBEF's single most identifiable copy signature is **committing hard to a campaign theme and letting the metaphor propagate through every line**. When the theme is "Come Sail Away," the bar becomes the *Galley*. When it's "Boots on the Beach," the best-dressed award becomes the *Saddled Up in Style Award*. This isn't decoration — it's the brand mechanic that makes HBEF copy unmistakable.

This file documents the theme lexicons for the events HBEF has run, and a framework for extending a new theme correctly.

---

## The framework: 4 layers of theme extension

When a theme is set, extend it through these four layers in priority order:

1. **Event title + subtitle** — the theme name itself
2. **Agenda / schedule labels** — re-name every time block in the theme's voice
3. **Awards / categories / honors** — every contest or callout gets a themed name
4. **CTA verbs + sign-offs** — "Set sail with us" instead of "Join us"

If you only have 30 minutes to write copy, get layer 1 right. If you have a full afternoon, push it through all four.

## Building a new theme

When HBEF picks a new theme (typically by the Hearts Co-Chairs in fall for the spring event), the process is:

1. **Pick a metaphor world** — usually evocative, hospitable, and unifiable (sea voyage, country fair, garden party, masquerade, golden hour)
2. **List the 8–12 metaphor-rich nouns and verbs** in that world (sail, mast, anchor, crew, voyage, port, deck, captain, etc.)
3. **Map each to an event element** (see tables above for the mapping pattern)
4. **Commit to using the themed term everywhere**, even in administrative copy — agenda items, ticket descriptions, auction-donation pleas, parking instructions

The test for whether a theme is well-extended: **could you skim 5 random sentences from the event page and immediately know the theme without seeing the title?** For "Come Sail Away," the answer is yes. That's the bar.

---

## Counter-examples: don't do this

- **Mixing two themes** — if it's "Come Sail Away," don't drop in a cowboy reference for variety.
- **Forced over-extension** — not every word needs a themed swap. "Tickets" can stay "tickets" if the themed alternative ("boarding passes") feels stilted in context. The natural-sounding 70% wins over the forced 100%.
- **Heavy-handed theme on serious posts** — Annual Giving check-delivery posts, IC updates, or in-memoriam content should stay neutral. Themes are for event copy, not financial reports.
- **Theme outliving the event** — once the event is recapped, retire the theme. Don't keep saying "Anchors Down" in posts that aren't about Hearts of Hermosa.

## Theme lexicon worksheet (for new events)

Use this template when starting a fresh theme:

```
EVENT: ___
DATE: ___
THEME NAME: ___
THEME WORLD: ___  (e.g. "yacht voyage" / "country fair")

KEY METAPHOR NOUNS (8–12):
1. ___
2. ___
…

EVENT ELEMENT → THEMED TERM MAPPING:
- Welcome / arrival      → ___
- Food + drink area      → ___
- Stage / DJ / band      → ___
- Auction (silent)       → ___
- Auction (live)         → ___
- Last call / closing    → ___
- Volunteers             → ___
- Co-Chairs / leadership → ___
- Donations              → ___
- Save-the-date verb     → ___
- Goodbye / sign-off     → ___

THEMED EMOJI ACCENTS:
- Primary: ___ (one nautical/western/garden emoji)
- Secondary: ___ (one that doubles for the event vibe)
- Sparing: ___ (one for max-celebration moments only)
```

Fill it in once; then every post pulls from this sheet rather than reinventing.
