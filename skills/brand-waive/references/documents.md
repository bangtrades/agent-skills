# WaiveLabs — Documents Spec (.docx / .pdf)

Pairs with the **`docx`** and **`pdf`** skills (mechanics). Reports, proposals, engagement letters,
SOWs, memos, one-pagers, case studies, guides. US Letter (8.5×11in), 1in margins.

## Master look

- **Header band:** primary horizontal logo lockup top-left; document type + date top-right, muted
  Sora caps. A thin `--blue` rule under the header.
- **Footer:** "WaiveLabs.ai · confidential" (when applicable) left, page number right, Inter 9pt
  `--ink-30`.
- **Type:** title **Sora** 700 ~26–30pt; H2 Sora 600 ~16pt with a short orange or blue underline;
  H3 Sora 600 ~12.5pt; body **Inter** 10.5–11pt, 1.4–1.5 line spacing, `--ink-body` `#1E2435`.
  Office fallbacks: Sora→Montserrat, Inter→Inter/Arial.
- **Color use:** blue for headings/links/rules; orange for a single highlighted figure or callout
  bar; mist/cream fills for callout boxes; near-black body. Keep it calm and white-spaced — these
  read as premium consulting documents, not flyers.
- **Components:** callout box (left `--sun` border, cream fill) for the key insight; a "by the
  numbers" KPI strip (Inter tabular figures + delta); clean tables with a `--dawn-2` header row,
  hairline `--line` rules, no heavy gridlines.

## Document patterns

- **Proposal / SOW:** cover (mark + client + date) → the 90-day offer → scope/deliverables →
  phased timeline (Data→…→Graduation) → programs as **tiers** (dollar figures ONLY in a private,
  explicitly-quoted version) → terms → email CTA. Engagement docs may state price when the doc is a
  direct, named quote — never in anything that circulates publicly.
- **Report / case study:** title + dek → executive summary (3–4 sentences, the "so what") →
  findings in prose (no bullet spam) → a KPI strip → recommendation → next step. Real numbers,
  honest framing, no claimed results we don't have.
- **One-pager:** single sheet — headline promise, 3 proof points, one KPI, email CTA. Dawn or Dusk.
- **Letterhead / memo:** header band + body in Inter; signature block; "WaiveLabs.ai" footer.

## Voice in documents

Plain English first, term second; prose over bullets for narrative sections; concrete numbers;
end on a clear next step (an email, not a calendar). No pricing on public collateral. Verify the
*i* in WaiveLabs. (See `voice.md`.)

Montserrat/Inter fallbacks. The site's OG cover (`og-cover.png`) is the brand's reference visual if
